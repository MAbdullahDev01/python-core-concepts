# Syntax & Types
---

## Variables
- A location where a value is stored in data memory
- Value can change
- snake_case naming convention is used

## Constants
- A location where a value is stored in data memory
- Value cannot change
- UPPER_SNAKE_CASE naming convention is used
---
## Data Types

### Integer
- A whole number

### Float
- A number containing a decimal

### Complex
- Numbers with real and imaginary parts

### String
- A series of alphanumeric characters

### Boolean
- True or False
---

## Lists
- A single variable used to store multiple values
- [] brackets are used
- starts at index 0
- mutable

## Dictionary
- A single variable used to store multiple key : value pairs
- {} brackets are used
- mutable

## Sets 
- A single variable to stored multiple values
- immutable
- no duplicate values
- unordered

## Tuples 
- A single variable to stored multiple values
- ordered
- immutable
- allows duplicates
- () brackets used
---

## F-Strings
- formatted strings
- f used before ""
- {} used to represent a variable
