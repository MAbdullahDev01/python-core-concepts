# Control Flow
---

## if/elif/else
- Conditional statement used to check if a certain condition is true
- statements under if are ran when condition is true
- elif is used to check multiple conditions
- else is ran when no condition was true
---

## Loops

### For
- Count controlled loop

### While
- Pre-conditional loop
---

## Comprehensions
- Used to write a shorter version of a loop and/or condition within one line
---

## Walrus Operator
- Used to assign and return value
- Can be used in loops/conditionals since it returns value
