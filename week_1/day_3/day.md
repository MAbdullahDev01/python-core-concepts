# Functions
---

## Arguments
- The actual value passed to a function

## *args
- the single * means any amount of input will be converted into a tuple
- comes before **kwargs if both used in a function

## **kwargs
- the double ** means all the arguments will be treated as objects in a dictionary

## Default Values
- set a predefined value for a parameter which can change if the user enters an argument

## Scope
- the textual region of a program where a specific variable or name is directly accessible and recognized

## Recursion
- a function calling itself after breaking down a larger problem into sub-problems
